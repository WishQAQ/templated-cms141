深圳设计师浩子和77的11天成都-稻城自虐骑行

时间：2014-02-06 12:28编辑：xunmoban.com

刚要成熟，便要老去，有些人，真的不必再等，有些事，必须现在就要去做！腾讯ecd中心同学疯浩子和77同学11天成都至稻城自虐骑行，带来骑行中的视觉大片和艰辛的点滴，而这次旅行的意义是什么？听听对他们的采访。
浩子：腾讯移动终端设计组总监　微博 博客

77：腾讯电商用户体验设计部成员

感谢腾讯ecd友情供稿 点击查看他们的博客

2012年十一长假，抓着秋天的尾巴，浩子和77一拍即合决定骑单车从成都骑到亚丁，去之前浩子特意看了《转山》，说太假了人怎么会骑成那个鸟样，然后浩浩荡荡地出发了。

这次长途骑行的目的是什么？

浩：减肥。 7 ：通过近乎自虐的方式，发泄工作压力，生活压力。 。。。。。。

难道不是那种有些事这次不做就再也不会做了的情怀吗？

浩:至少三分之一是减肥（很执着 没听进去），也是想有一次长途骑行的经历，觉得自己体力还行。 这次旅途第一个失望的地方是，路上空气太差，很多灰，很多大货车的尾气。地面很脏。

还觉得转山夸张吗？

太真实了，比转山还TM惨。

觉得最美的景色在哪里？你们看到雪山了吗？

浩：看到了，看到日照金山。那个还好吧。

还好？!!

浩：是很美，但都是想象中之内的美。我经常滑雪，见过不少雪山，但那些都是可以玩的，不像这个，你只能远远地看着。有些人旅游是为了旅途，而有 些人是为了特有的目的去的。我一般都是想去玩，期望有很多好玩的，或者是说非常好的景色。对我来说不能接近的景色，太遥不可及了。

我觉得最美的景色是在一个镇子去稻城的路上，那天真正意义上的进藏。 出了四川境，开始没什么人，绕着村庄，拍了很多村庄里的小孩。

7：我觉得是最后的那几天， 从来 理塘 至 稻城 ，那段路几乎没有人住，五颜六色的山与树林，还有清晰见底的小溪水。 非常适合露营，很想再骑那段路一次，带上帐篷。

最难走的路？

浩：折多山，前面还有个二郎山也很难走，我们本来觉得二郎山最难走了结果遇到了折多山哈哈哈哈。折多山是4800多米，单车都冻住了，刹车轮胎也冻 住了。最后几步路都是推着轮子在路边滑行。推到山顶上。这是两次感到害怕的时刻。每两座山之间，路程没有多远，几十公里，但是骑车上去就很慢。 山上一到了晚上全是黑的，用一个小手电照着，你必须得一直骑，骑下山。那天晚上没有星星月亮。 对了下午四点多的时候，在折多山上碰到一个藏民，在雪山上有个小屋子，不然77就冻得不行了。藏民很友好，让我们进屋。

骑到离山顶还有十几公里，那时候还有人在看雪景，晚上就比较可怕，要躲货车。 折多山骑了十几个小时。我们每天都要翻一座山。

7：最杯具的是爬雪山的?时候，我那天有高原反应，带来的是一次一次的希望与绝望。晚上8点多雪正在下，没有路灯路很黑、很冷、很饿、很渴，问第1个 人说距离山顶有4公里，第2个人说快到了，第3个人说还挺远的你们不要死撑了建议你们沿途返回第二天才爬山吧，第4个人说这个不太了解多少公里哦开 车20分钟。想放弃把路过的车已经截停了，准备上车的时候犹豫了，再坚持一下应该就能到顶了，最后我们没有坐车撑到了山顶。

途中最触动你的地方？

浩：骑行应该是更接近大自然的，但是骑行的过程中往往会忽略很多，因为每天你都要赶时间，要在规划时间内爬过一个山头。每一次停，你都要算好我能不能在天黑之前骑到目的地，和夜晚做赛跑。我们在山顶上讨论写遗嘱的事情，其实我有写遗嘱。

可以亮出来吗？

太隐私了，不行。＝３＝

骑单车的时候脑袋一整天都在想事情，这点很好，我一直在思考人生。 每天都在写日记，类似有一点忏悔录的感觉，每天都在反省自己，还是很不一样的体验，对自己人生还是有好处的。 我想体验一些比较真善美的人生。

这段骑行完会更珍惜很多东西，会珍惜更多以前你认为可以随意抛弃的东西。 人的要求并不是特别多，你在路上只是需要饼干和水。

7：遗嘱，第一次骑行的时候有写，遗嘱有个问题就是怕死了后没有人知道，所以我是在邮箱里设了个定时发送，能安全回来再取消。浩子这次好像就写了， 我没有写了，但每次出门必买保险。自虐行最大收获就是，得到一个自我安慰的借口，经历过这么艰苦和极端的环境都能熬过去，现在遇到困难会想想，有 比较骑行更痛苦的事吗？ 也就是因为这一句话导致我经常加班，也觉得很快乐。 （他在哭！）

有趣的事？

浩：到了亚丁，我们特意去野炊。找了块真正的野山，在小溪边上，买了几个土豆，煮了方便面，77自己编了草鞋穿，捡了破锅，还做了锅刷。 那天是最爽的一天。 还有在隧道里裸奔，和在山顶上晨便。沿路的司机都在拼命按喇叭，特别是女司机（得意），那时候觉得什么都不在乎。

7：野炊 裸奔。。。不应该叫晨便， 有一首诗的一句叫 飞流直下三千尺 ，我们就是站在瀑布顶上尿尿的人了。

77你编草鞋的功夫很了得啊！

那是看电视节目学的，刚好有机会就学习做了一下，对自己的第一次作品感觉还挺满意的。

艳遇没有吗？

浩：在路上没有遇到一个美女，这是最遗憾的地方。（发自内心的叹息） 7 ：很可惜没有~ 就连酒吧也没有机会去，这是个败笔。

你最需要的什么这一路上？

7 ：我有点贪心需要两个东西 1. 女朋友，与她分享眼前那些美好的事物。 2. 更多的时间，这次太赶了导致一些路程还坐了车。 浩：女人!!!（发自内心的诚实）

"有人说你永远拍不下最美的景色，因为最美的景色是拍不下来的，可是当你回来看照片的时候，还是会庆幸当时毫不犹豫按下了快门。"

"心存美好，必将途遇美好。"

他们的路线：

2012年9月22日~10月7日 成都至稻城亚丁。全程约770KM，时间共16天，其中用11天时间骑行，2天机动安排，3天作大巴往返回之用。

行程：

Day1. 雅安 → 始阳 → 天全 → 新沟（海拔1310） 分段里程：90公里

Day2. 新沟 → 二郎山隧道（海拔2170） → 泸定(海拔1330) 分段里程：54公里

Day3. 泸定 → 瓦斯 → 康定(海拔2395) 分段里程：50公里

Day4. 康定 → 折多山口(海拔4298) → 新都桥(海拔3630) 分段里程：75公里

Day5. 新都桥 → 高尔寺山口(海拔4270) → 雅江（海拔 2530）分段里程：72公里

Day6. 雅江 → 剪子湾山口(海拔4450) → 119道班（海拔 4260）分段里程：52公里

Day7. 119道班 → 卡子拉山（海拔4440） → 理塘(海拔3968) 分段里程：110公里

Day8. 理塘 → 扎嘎山（海拔4130米） → 兔儿山（海拔4640米） → 75道班（海拔4360米）分段里程：75公里

Day9. 75道班 → 海子山（4610米） → 桑堆乡（海拔3940） → 稻城 分段里程：70公里

Day10. 稻城 → 日瓦（海拔3000+） 分段里程：34公里

Day11. 日瓦 →亚丁（海拔3000+） 分段里程：77公里
